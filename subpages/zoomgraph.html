zoom grapher





execute with nodejs. This automatically draws an image using the zoom comment tool when in screenshare. Remember to resize the image to appropriate size because every pixel line will correspond in a line drawn by the bot! required packages:
  • robotjs
  • image-js

  • zoomgraph.js ⤓

    usage:
    node zoomgraph.js [image file]
    -w draws white last, usually recommended
    -v displays debug info
    -s # defines the line width as #, 2 is default (zoom line widths: 1, 2, 4, 6)
    -d # sets the delay after releasing/pressing the mouse as #ms, default is 5ms (increase when lines are not drawn or at the wrong place)
    -h show this dialog

    example: node zoomraph.js bob.png -w -s 1

    after the program starts it will prompt you to first put your mouse on the button that will change the color, put it as close to the centre as you can. then put your mouse where you want the top left corner to be. as a failsafe you can quickly swipe your mouse to the right to abort
