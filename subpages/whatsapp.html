whatsapp bots

these bots are running on your device, so you won't be able to use them and all messages sent from the bot are seen as yours. to run those bots you need to execute them using nodejs. after executing the script you will have to scan the generated qr-code with whatsapp Settings > Whatsapp Web/Desktop > add Device

required packages:
  • whatsapp-web.js
  • qrcode-terminal

  • pollbot ⤓

    this allows people to create polls in chats you are part of and vote on them by sending commands! command list
